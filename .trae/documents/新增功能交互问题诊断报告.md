## 🎯 核心问题总结

### 问题 1：文件选择器未隐藏 ❌
**影响所有 4 个模块**

所有新增模块的 `<input type="file">` 都没有设置 `style="display:none"`，导致：
- 文件选择器直接显示在页面上
- 用户可能直接点击 input 而不是 upload zone
- UI 混乱

**需要修复的元素：**
- `page-manager-file` (L1085)
- `encrypt-file` (L1142)
- `decrypt-file` (L1190)
- `page-number-file` (L1230)
- `sig-upload-file` (L1335)
- `sig-pdf-file` (L1358)

---

### 问题 2：初始化时机可能有问题 ⚠️
**影响所有 4 个模块**

`main.js` 中所有 `init*()` 函数在 DOMContentLoaded 事件中调用，但如果：
- 某些元素在 modal 中，modal 未打开时元素可能存在但未渲染
- 浏览器可能延迟加载某些元素

**建议：** 添加 `console.log` 调试信息，确认元素是否正确获取

---

### 问题 3：缺少调试信息 🔍
**影响所有 4 个模块**

所有模块都没有 `console.log` 调试信息，无法确认：
- 初始化是否成功执行
- 元素是否正确获取
- 事件是否正确绑定

---

### 问题 4：加密模块权限复选框未使用 ⚠️
**影响加密模块**

HTML 中定义了权限复选框：
- `encrypt-allow-printing` (L1168)
- `encrypt-allow-copying` (L1169)
- `encrypt-allow-modifying` (L1170)

但 `encrypt.js` 中的 `encryptPdf()` 函数没有读取这些复选框的值。

## 🔧 具体修复方案

### 修复 1：隐藏文件选择器（P0）
在 `index.html` 中为所有 `<input type="file">` 添加 `style="display:none"`：

```html
<input type="file" id="page-manager-file" accept=".pdf" style="display:none;">
<input type="file" id="encrypt-file" accept=".pdf" style="display:none;">
<input type="file" id="decrypt-file" accept=".pdf" style="display:none;">
<input type="file" id="page-number-file" accept=".pdf" style="display:none;">
<input type="file" id="sig-upload-file" accept="image/png,image/jpeg,image/jpg" style="display:none;">
<input type="file" id="sig-pdf-file" accept=".pdf" style="display:none;">
```

### 修复 2：添加调试信息（P1）
在每个 `init*()` 函数开头添加：

```javascript
export function initPageManager() {
  console.log('[PageManager] Initializing...')
  const uploadZone = document.getElementById('page-manager-upload')
  const fileInput = document.getElementById('page-manager-file')
  console.log('[PageManager] uploadZone:', uploadZone)
  console.log('[PageManager] fileInput:', fileInput)
  // ...
}
```

### 修复 3：加密模块权限复选框（P2）
在 `encryptPdf()` 函数中读取复选框值：

```javascript
const allowPrinting = document.getElementById('encrypt-allow-printing')?.checked ?? true
const allowCopying = document.getElementById('encrypt-allow-copying')?.checked ?? true
const allowModifying = document.getElementById('encrypt-allow-modifying')?.checked ?? true
```

### 修复 4：初始化时机优化（P3）
在 `main.js` 中确保 DOM 完全加载后再初始化：

```javascript
window.addEventListener('DOMContentLoaded', () => {
  // 添加延迟确保 DOM 完全渲染
  setTimeout(() => {
    initPageManager()
    initEncrypt()
    initPageNumber()
    initSignature()
  }, 100)
})
```
